PC Installation

List of Items

  • CPU
    • CPU could be either Intel or AMD
  • Motherboard
    • If choosing Intel as CPU. Make sure the Motherboard is compatible with Intel CPU.
  • Memory
    • Depends on your needs. If you are a heavy video editor. 32GB of RAM is necessary. Otherwise, 16GB is enough.
  • Storage
    • Recommend using SSD for since it will increase PC's performance
  • Graphic Card
    • Using Nvidia Graphic Card if you care about playing game.
    • Using AMD Graphic Card if you care about Video Rednering performance.
  • Power Supply (PSU)
    • 800W PSU is good enough.
  • Computer Case
    • Depend on your Motherboard size, choose the one that will fit your Motherboard.
  • Windows Key

Steps

  1. Put CPU on the Motherboard first.
  2. Install the included heatsink for the CPU.
  3. Put the Motherboard into the Computer Case. Then, securing it by using the provided scews.
  4. Install RAM and Storage.
  5. Install Power Supply
  6. Booting up the Computer and install the Operating System (OS) for it. Then, install softwares you like to use.
  7. Enjoy!!
